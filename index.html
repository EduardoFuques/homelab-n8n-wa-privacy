Política de Privacidad – Homelab n8n WA

Última actualización: 14/12/2025

Esta Política de Privacidad describe cómo Homelab n8n WA (en adelante, el “Servicio”) trata la información al interactuar con usuarios a través de WhatsApp Cloud API de Meta, con el fin de disparar flujos automatizados en n8n.

1. Responsable del tratamiento

Responsable: Eduardo Fuques Co. (Eduardo Fuques)

Email de contacto: edufuques@gmail.com

2. Alcance

Esta política aplica al tratamiento de datos generado por:

  • Mensajes entrantes enviados por usuarios al número asociado a WhatsApp Cloud API utilizado por el Servicio.
  • Metadatos necesarios para procesar dichos mensajes (por ejemplo, identificadores de conversación/usuario provistos por la plataforma).

3. Datos que se procesan

El Servicio puede procesar, de forma transitoria, los siguientes datos:

  • Contenido del mensaje enviado por el usuario.
  • Número de teléfono del usuario (o identificadores equivalentes proporcionados por WhatsApp/Meta).
  • Fecha y hora del mensaje y metadatos técnicos mínimos necesarios para el enrutamiento y ejecución del flujo.

4. Finalidad del tratamiento

Los datos se procesan exclusivamente para:

  • Recibir mensajes entrantes vía WhatsApp Cloud API.
  • Disparar y ejecutar flujos automatizados en n8n basados en esos mensajes.
  • Responder al usuario en caso de que el flujo lo requiera.

5. Retención de datos

El Servicio está diseñado para no almacenar mensajes ni su contenido de forma persistente.

  • No se mantiene una base de datos propia destinada a guardar historiales de chat.
  • Los mensajes se procesan de manera transitoria para ejecutar automatizaciones.

Nota: Pueden existir registros técnicos (logs) de infraestructura o de terceros (por ejemplo, de Meta o del hosting/red) que estén fuera del control directo del Servicio.

6. Compartición de datos y terceros

El Servicio utiliza WhatsApp Cloud API, provista por Meta Platforms, Inc., para recibir y enviar mensajes.

  • Al usar WhatsApp, el usuario también está sujeto a las políticas y términos de Meta/WhatsApp.
  • El Servicio no vende datos ni los comparte con terceros con fines publicitarios.

7. Base legal

El tratamiento se basa en:

  • Consentimiento y/o acción del usuario al iniciar el contacto y enviar mensajes al canal de WhatsApp utilizado por el Servicio, y
  • La necesidad de procesar esos datos para prestar la funcionalidad solicitada (automatización basada en mensajes).

8. Seguridad

Se aplican medidas razonables para proteger la información durante el procesamiento (por ejemplo, controles de acceso al entorno, segmentación de red y buenas prácticas operativas). Sin embargo, ningún sistema es completamente infalible.

9. Derechos del usuario

El usuario puede solicitar información o ejercer derechos relacionados con sus datos (por ejemplo, acceso, rectificación, eliminación) contactando a:

edufuques@gmail.com

Dado que el Servicio no retiene mensajes, es posible que no exista información persistente que eliminar más allá de registros técnicos externos al Servicio.

10. Menores de edad

El Servicio no está dirigido a menores de edad. Si un tutor considera que un menor ha provisto datos, puede contactar al email indicado.

11. Cambios a esta política

Esta política puede actualizarse para reflejar cambios técnicos u operativos. La fecha de “Última actualización” indicará la versión vigente.
